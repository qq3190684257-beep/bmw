# 0.3.0 Native Hybrid

- Keeps the game-native white trajectory and three-reflection bank preview.
- Applies `SettingData.pocketCueReflectNum = 3` from the Unity main thread.
- Reuses the 0.2.3 post-collision extension model.
- Draws only the yellow object-ball continuation and cyan dashed cue-ball continuation.
- Removes the duplicate orange approach polyline.
- Hides the menu and force HUD by default; the overlay window remains click-through.
- Does not modify shot force, input, ball state, or network traffic.
